MCP - протокол для взаимодействия LLM с сервисами

2025-03-26 :: Иван :: 1 min read (101 words)
#ai  #ml  #agency 

За два дня дважды в разных местах попалось упоминание MCP (Model Context Protocol) (открытая технология Anthropic).

Что это:

  • https://modelcontextprotocol.io/introduction ссылка на введение в протокол. Может смущать наличие и host, и client, и server: как я понял почитав сайт, основная единица это server, который отвечает на запросы, которые client делает (по указке hostа).
  • https://mcp.so/ тут много примеров MCP.
  • https://habr.com/ru/articles/893482/ - статья про MCP (краткая обзорная часть и примеры кода).

См. также LSP :-) (который во многом похожее делает).

(С точки зрения AI Safety… пф-пф-пф. Ну, клиент может обрубать некорректные запросы от хоста. Можно сделать сервер-gateway, который будет ограничивать права в зависимости от).
